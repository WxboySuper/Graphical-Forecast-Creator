# v1.2.0 Launch Notes

## Release Summary
Version 1.2.0 highlights a new undo/redo workflow on the forecast page so forecasters can iterate on polygons with a safety net. This release also includes internal branch-porting cleanup to keep maintenance PRs smaller and easier to review.

## Support Discord Message
Graphical Forecast Creator v1.2.0 is now ready for release.

The headline change in this update is forecast-page undo/redo. You can now reverse recent drawing edits from the toolbar or with standard shortcuts:

- `Ctrl/Cmd+Z` to undo
- `Ctrl/Cmd+Y` to redo
- `Ctrl/Cmd+Shift+Z` to redo

This should make it much easier to refine outlook boundaries without having to redraw polygons after a mistake.

This release also includes some behind-the-scenes maintenance improvements to keep our branch porting workflow cleaner going forward.

If you spot anything odd after the release goes live, please drop it here in the support server with screenshots and the day/outlook type you were editing.
