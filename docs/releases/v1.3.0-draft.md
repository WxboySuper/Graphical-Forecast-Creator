# v1.3.0 Release Notes

**Target Release:** March 2026

## Highlights

This release brings major quality-of-life improvements for forecast creation and verification workflows.

## New Features

### Ghost Overlays
Toggle previous outlooks as semi-transparent overlays on the map for easy alignment and comparison. Access via the toolbar in the forecast view.

### Per-Day Undo/Redo
Fixed undo/redo history so it now persists correctly when switching between forecast days. Your edit history is preserved across day transitions.

### Verification Storm Report Windows
Added support for loading both today's and yesterday's storm reports in the Verification panel, making it easier to compare current vs. recent storm activity.

### Alert Banner System
Added a configurable alert banner system that can display info, warning, or error messages to all users. Managed via `public/alert-banner.json`.

## Migration Notes

No breaking changes. The alert banner is disabled by default.
